# CRIANDO PROJETO
```
npx nuxi@latest init frontend
```

## Modulos instalados
* __ui__: Esse módulo já vem com um componente `<UModal>` pronto, além de botões, inputs e tabelas bonitas baseadas em Tailwind CSS. Fazer um modal do zero (HTML/CSS/JS) que fecha ao clicar fora e centraliza na tela é chato e consome tempo. Com esse módulo, é uma linha de código.

* __eslint__: Ele vai avisar se você deixar variáveis não usadas ou cometer erros de sintaxe bobos antes de você entregar. Garante que seu código esteja "limpo" e padronizado.

* __icon__: Em vez de baixar imagens SVG na internet, você só escreve `<Icon name="heroicons:pencil" />` e o ícone aparece. Facilita muito a interface do CRUD.

## Rodando projeto:
```
npm run dev
```
